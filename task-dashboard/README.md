# Task Manager App
## Overview

A simple and interactive Task Manager web application built with React and TypeScript. This app allows users to create, edit, delete, filter, search, and sort tasks by due date. It also supports light/dark mode with theme preference saved in localStorage.

## Features
* Add, Edit, and Delete Tasks – Keep track of your tasks easily.
* Task Filtering & Search – Filter tasks by status, priority, or search keywords.
* Sort by Due Date – Quickly see which tasks are due soonest.
* Dark/Light Mode – Toggle between themes, and the preference is saved.
* Persistent Storage – All tasks are saved in localStorage for persistence across sessions.
* Responsive Layout – Works well on desktop and mobile devices.

## Technologies Used
* React (with Hooks)
* TypeScript
* Bootstrap for layout and styling
* LocalStorage for data persistence

## Usage
* Add a Task: 
   Fill in the task form (title, description, status, priority, due date) and submit.

* Edit/Delete a Task:
   Use the Edit or Delete buttons on each task card.

* Filter/Search Tasks:
   Filter by status (Pending, In Progress, Completed)
   Filter by priority (Low, Medium, High)
   Search for task title

* Sort by Due Date
   Click the Sort by Due Date (Soonest First).

* Toggle Theme:
   Click the Dark Mode / Light Mode button to switch themes.
